# Design System

Version: 0.2

> This document defines the visual identity of the Splinters of Faith
> soundtrack website. It focuses on **design philosophy** rather than
> implementation details. Any technology used to build the site should
> preserve this vision.

------------------------------------------------------------------------

# Design Statement

The website should feel like opening the **Collector's Edition
soundtrack companion** to a fantasy RPG.

Visitors should not feel like they are browsing a streaming service.

They should feel like they are exploring a world through its music.

Every page should invite exploration.

Every collection should feel like entering a new region.

The music should remain the heart of the experience.

------------------------------------------------------------------------

# Overall Style

Primary inspiration:

-   Final Fantasy
-   Modern JRPG companion sites
-   Premium fantasy art books

The site should be:

-   Elegant
-   Atmospheric
-   Warm
-   Premium
-   Easy to navigate

Avoid:

-   Corporate design
-   Generic streaming interfaces
-   Busy layouts
-   Excessive ornamentation

------------------------------------------------------------------------

# Website Philosophy

Navigation should feel like moving through a world instead of browsing
folders.

Major collections become destinations.

Examples:

-   Main Suite
-   Forest
-   Swamp
-   Mountains
-   Characters
-   Tavern Music
-   Stingers

Visitors should feel like they are *traveling*, not changing pages.

------------------------------------------------------------------------

# Typography

Use two font families.

## Display Font

Reserved for:

-   Hero banners
-   Collection titles
-   Major headings

Characteristics:

-   Elegant
-   Fantasy-inspired
-   Highly readable
-   Used sparingly

## Body Font

Used for all supporting content.

Characteristics:

-   Modern
-   Clean
-   Extremely readable

Typography should enhance atmosphere without sacrificing clarity.

------------------------------------------------------------------------

# Artwork

Artwork is the single most important visual element of the website.

Every major collection should feature unique hero artwork that
immediately establishes the identity of the page.

Artwork style:

-   High-end anime fantasy
-   Painterly
-   Cinematic lighting
-   Rich environments
-   Premium key-art quality

Artwork should answer the question:

> "Where am I?"

before the visitor reads a single word.

## Growing Artwork Library

The artwork library is expected to grow over time alongside the
soundtrack.

The architecture should assume that:

-   New collections will receive new artwork.
-   Existing artwork may be replaced with improved versions.
-   Additional artwork (banners, thumbnails, backgrounds, character
    portraits, etc.) will be added later.

The design should gracefully accommodate expansion without requiring
redesigns.

------------------------------------------------------------------------

# Color Philosophy

The website should not have one universal palette.

Instead, each collection inherits its own visual identity.

Examples:

Forest - Deep greens - Warm gold - Natural wood

Mountains - Blue-gray - Stone - Snow

Swamp - Olive - Fog - Blue-green

Desert - Amber - Sandstone - Warm sunlight

Characters - Palette inspired by the featured character artwork.

Changing collections should feel like entering a new environment.

------------------------------------------------------------------------

# Interface

Use soft rounded corners.

Employ warm, fantasy-inspired glass effects where appropriate.

Glass should frame the content---not become the content.

Artwork always remains the visual focus.

------------------------------------------------------------------------

# Motion

Animation should be subtle and purposeful.

Examples:

-   Gentle fades
-   Soft hover effects
-   Small parallax movement
-   Smooth page transitions

Avoid excessive animation.

The website should feel alive without demanding attention.

------------------------------------------------------------------------

# Audio Player

The player is persistent.

Playback should continue while navigating.

Desired playback modes:

-   Play Entire Soundtrack
-   Narrative Order
-   World Order
-   Shuffle

The player should always remain accessible without dominating the page.

------------------------------------------------------------------------

# Hero Sections

Every major page begins with immersive artwork.

Recommended layout:

Hero Artwork

↓

Collection Title

↓

Short Description

↓

Play All

↓

Track Listing

The visitor should immediately feel they have entered a new location.

------------------------------------------------------------------------

# Emotional Design

Every collection should evoke a specific emotional response.

Examples:

Home - Wonder - Curiosity

Main Suite - Beginning an adventure

Forest - Peace - Hope - Quiet exploration

Mountains - Grandeur - Determination - Scale

Swamp - Mystery - Unease - Discovery

Tavern - Warmth - Fellowship - Rest

Characters - Personal connection

Stingers - Energy - Surprise

Every design decision should reinforce the intended emotion of its
collection.

------------------------------------------------------------------------

# Design Principles

1.  Artwork comes first.
2.  Music never stops.
3.  The world is the navigation.
4.  Every collection has its own visual identity.
5.  Atmosphere is more important than decoration.
6.  Consistency is more important than novelty.
7.  The interface should support the music, never compete with it.
8.  Expansion should always be anticipated.

------------------------------------------------------------------------

# Guiding Question

Before implementing any feature, ask:

**Does this help the visitor feel like they are exploring the world of
Splinters of Faith through its music?**

If the answer is no, reconsider the design.
